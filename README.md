# Spyder-Tahminleme-Projesi
500 gözlem ve 40 değişkenden oluşan sağlık
verisiyle tahminleme modeli geliştirildi. %7 lik hatayla tahmin modeli en başarılı model seçilmiştir.

Veri ön işleme adımında eksik değer yönetimi ve
kategorik değişken dönüşümleri uygulandı.

K-Means ile hasta profilleri segmentlere ayrıldı.
Lojistik Regresyon ve Random Forest modelleri
kuruldu.

Özellik seçimi sonrası Random Forest modeliyle
%98,9 doğruluk elde edildi.

En etkili değişkenler: FEV1/FVC, yaş, cinsiyet, sigara
kullanımı olarak bulundu.
